# browser.isolate&#x20;

* **Type:** `boolean`
* **Default:** the same as [`--isolate`](/config/#isolate)
* **CLI:** `--browser.isolate`, `--browser.isolate=false`

Run every test in a separate iframe.

::: danger DEPRECATED
This option is deprecated. Use [`isolate`](/config/#isolate) instead.
:::
